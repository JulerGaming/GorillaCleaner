# GorillaCleaner
GorillaCleaner is a bot that protects servers from people that engage in gorilla tag copies and cheat servers.

## Features

- Detects and removes gorilla tag copies
- Identifies and manages cheaters
- Real-time server protection
- Easy configuration and deployment

## Installation

1. Clone the repository
2. Install dependencies: ```npm install```
3. Configure your bot token in ```.env```
4. Run: ```npm start``` OR ```pm2 start ecosystem-config.js```

## Usage

Add GorillaCleaner to your server and it will automatically monitor for violations.

## Support

For issues and questions, please open an issue on GitHub.
## Additional Features

- Automatically bans users found in cheat servers
- Removes members detected in gorilla tag copy servers
- Maintains a database of known violation servers
- Supports PM2 for production deployment
